# Sort Number App

## Usage

1. Clone This Repository

```bash
git clone https://github.com/KucingOyenBarbar/vite-sort-number-app.git
```

2. Install Required Packages

```bash
npm run install
```

```bash
npm run build
```

```bash
npm run preview
```

or

```bash
yarn install
```

```bash
yarn run build
```

```bash
yarn run preview
```
